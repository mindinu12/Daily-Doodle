# 🎯 Daily Doodle

Welcome to **Daily Doodle (Macondo Edition)**! This is a hyper-focused, minimal retro dashboard built with vanilla web technologies. It is designed to keep you productive, test your reaction speed, and give you a spark of style inspiration every single day.

---

## 🕹️ App Features

*   **Daily Quests:** A clean checkpoint grid to track your daily programming targets and habits.
*   **Catch the Oodle:** A fast-paced, 30-second reaction mini-game featuring responsive target shifting and local high-score persistence.
*   **Style Roll:** An inspiration layout powered by dynamic image loading pipelines.
*   **Neo-Brutalist Design:** High-contrast borders, solid black shadow depth, and monospaced typography optimized for scanning.

---

## 🚀 Getting Started

### Project Files
Your workspace must contain these three core files in the same directory:
*   `index.html` — The structural page hierarchy and layout nodes.
*   `index.css` — The Neo-Brutalist grid styling, UI boundaries, and responsive breakdowns.
*   `app.js` — The game loop mechanics, image state engine, and memory handlers.

### Execution
To launch the system locally without any external build chains or dependencies:

1. Clone or download your repository.
2. Double-click `index.html` to execute the code inside any standard web browser viewport.
